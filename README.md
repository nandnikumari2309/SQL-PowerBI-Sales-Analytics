# SQL-PowerBI-Sales-Analytics

## 📌 Project Overview
This project demonstrates an end-to-end Sales and Customer Analytics Dashboard built using SQL Server and Microsoft Power BI.

The project involves:
- Designing SQL database tables
- Writing SQL queries for data analysis
- Creating analytical views
- Building an interactive Power BI dashboard
- Generating a project report

---

## 🛠️ Tools & Technologies
- SQL Server
- Microsoft Power BI
- SQL
- DAX
- Microsoft Excel
